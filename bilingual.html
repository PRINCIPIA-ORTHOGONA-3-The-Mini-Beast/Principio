0.2 · Notation

Conventions used throughout the book

Definition · Notation

The superscript in expressions such as gk is an ordinal label, not an exponent. Likewise, the symbols C, K, F, U, and T name operators in a chain rather than independent unrelated objects.

CKFUT = Compression → Constraint → Fold → Unfold → Time-circuit

Remark

When the same symbol appears in both Portuguese and English paragraphs, the meaning is intended to be identical. The bilingual layout changes the language of exposition, not the mathematical content.

0.3 · Taxonomy

The g-series

Series Definition

The g-series is a finite ordered set of regime labels used to classify qualitative stages of a dm3 system. In the source text, the labels are g0, g2, g6, g33, and g64.

Português

A série g não descreve uma sequência numérica comum. Ela funciona como uma taxonomia de regimes, em que cada etiqueta nomeia uma condição estrutural distinta.

English

The g-series does not describe an ordinary numeric sequence. It functions as a taxonomy of regimes, where each label names a distinct structural condition.

0.4 · Core ideas

Definitions carried into the book

Definition 3.1 · Generative Operator Chain

The single-cycle generative operator is CKFU, and the full chain including the fifth operator is CKFUT. Compression reduces complexity, Constraint shapes form, Fold creates self-reference, Unfold releases latent potential, and Time-circuit closes the cycle.

Definition 3.2 · Stability Threshold g33

The index 33 is treated in the source as a heuristic stability threshold for regenerative coherence. Below that threshold, the system is conjectured to be fragile; at or above it, the organizing invariant is conjectured to persist across disruption.

Definition 3.3 · Circuit Saturation at g64

After 64 cycles, the system maps its entire possibility space. The return is not a simple loop; it is described as a spiral return, meaning that the source is enriched by one completed circuit.

Definition 3.4 · Spiral Return

The return is a new application of the operator at circuit scale. The action variable accumulates dissipation along the return, encoding the thermodynamic cost of the completed cycle.

0.5 · Central statement

Theorem T1 in Chapter 0 form

Theorem · Generative Time Circuit

In the operator algebra on the contact manifold, time is the generative circuit operator T, the fifth stage of the chain CKFUT. For systems that have completed at least the stability threshold, the spiral return is enriched rather than linear.

Reading Note

This chapter does not yet attempt to develop every proof detail. It prepares the reader for the theorem statement, the proof strategy, and the later technical sections that expand the argument.
